# Dotfiles

This home directory is managed by a bare git repo at `~/.dotfiles`.

- Use `dot` instead of `git` for all dotfile operations: `dot add`, `dot status`, `dot commit`, etc.
- Alias: `dot='git --git-dir=$HOME/.dotfiles --work-tree=$HOME'`

## Environment

- Shell: bash (via Homebrew)
- Package manager: Homebrew
- macOS (Apple Silicon)
